HTB CTF Write-Up: Fawn

This write-up documents the enumeration of an SMB service on a Windows target, focusing on identifying open ports, discovering available shares using smbclient, and accessing a misconfigured shared folder with a blank password. By enumerating the WorkShares share, it was possible to browse user directories and retrieve the final flag, demonstrating how improper SMB share permissions can lead to information disclosure.

Tools: ping, smbclient, Linux CLI
