# Paulo Melo | Junior Web Developer Portfolio
Personal portfolio website showcasing my projects and skills as a Junior Web Developer. Built with HTML, CSS, and JavaScript, following mobile-first principles. The site demonstrates responsive design, frontend and backend fundamentals, dark/light mode, bilingual support (PT/EN), and clean, maintainable code.

## Live Demo
You can view the portfolio online here:
[My Portfolio](https://pjsm-it.github.io/)

## Tech Stack
* HTML5, CSS3, JavaScript (ES6+) 
* Responsive Design (Mobile-First)
* Dark/Light Mode
* PT/EN Language Toggle
* CSS Animations and AOS.js
* Typed.js for hero section animations
* EmailJS for contact form
* Git & GitHub for version control

## How to Run Locally
1. Clone this repository:
`git clone https://github.com/pjsm-it/pjsm-it.github.io.git`
2. Open index.html in your preferred browser (Chrome, Firefox, Safari).
3. (Optional) Use VS Code and the Live Server extension for local development with auto-reload.

## FEATURES
* Responsive Design: Works on mobile, tablet, laptop, and desktop 
* Dark/Light Mode: Toggle theme with localStorage persistence
* Bilingual Support: PT/EN toggle stored in localStorage
* Interactive Projects Section: Hover effects, smooth scroll
* Contact Form: Works via EmailJS (no backend required)
* Optimized Assets: WebP images, lazy loading, lightweight fonts
* SEO Friendly: Meta tags, Open Graph, sitemap, robots.txt
